# genice-localstruct

A [GenIce](https://github.com/vitroid/GenIce) plugin to calculate local structure indices for water.

## Requirements

* [GenIce](https://github.com/vitroid/GenIce) >=0.23.

## Installation from PyPI

    % pip install genice-localstruct

## Manual Installation

### System-wide installation

    % make install

### Private installation

Copy the files in genice_bondtwist/formats/ into your local formats folder of GenIce.

## Usage

	% genice T2 -f localstruct > T2.local
